# Detecting Overconfident Errors in Deep Learning Models Using Calibration and Transformer-Based Approaches

**Author: Bhavesh Uttarkar**

## Overview

This project focuses on detecting overconfident errors in deep learning classification systems. Deep learning models can sometimes make incorrect predictions with very high confidence, which creates reliability and trust issues in real-world applications.

The project implements a complete two-stage pipeline using the CIFAR-10 image classification dataset. In the first stage, a Convolutional Neural Network (CNN) is trained for image classification. In the second stage, the softmax probability outputs generated by the CNN are analyzed to detect overconfident incorrect predictions.

The project compares three approaches for overconfidence detection:

* Confidence threshold baseline
* Multi-Layer Perceptron (MLP)
* Transformer-based detection model

The system also evaluates calibration quality using Expected Calibration Error (ECE) and generates multiple visualizations for confidence analysis and model reliability.

---

## Dataset

This project uses the CIFAR-10 dataset from torchvision.

CIFAR-10 contains 60,000 color images divided into 10 classes.

The dataset is automatically downloaded during execution.

The dataset is split into:

* Training set
* Validation set
* Test set

Basic normalization preprocessing is applied before training.

## Two-Stage Pipeline

### Stage 1: Image Classification

A CNN model is trained on the CIFAR-10 dataset for image classification.

After training, the model produces:

* Prediction probabilities
* Predicted labels
* Confidence scores

These outputs are used for calibration analysis and overconfidence detection.

---

### Stage 2: Overconfidence Detection

The second stage identifies cases where the CNN prediction is highly confident but incorrect.

An overconfident error is defined as:

* Prediction confidence above a threshold
* Prediction is incorrect

Three detection approaches are compared:

#### Threshold Baseline

Uses confidence thresholding directly on prediction probabilities.

#### MLP Detector

A Multi-Layer Perceptron learns patterns from the probability distributions.

#### Transformer Detector

A Transformer-based model analyzes relationships between class probability distributions for overconfidence detection.

## Calibration Analysis

The project evaluates model calibration using Expected Calibration Error (ECE).

ECE measures how closely model confidence matches actual prediction accuracy.

Reliability diagrams are also generated to visually analyze calibration quality.

---

## Purpose of the Project

The goal of this project is to move beyond simple classification accuracy and study model reliability and trustworthiness.

Detecting overconfident errors is important in sensitive applications such as:

* Healthcare
* Finance
* Autonomous systems
* Security systems

The project demonstrates how calibration analysis and deep learning-based detection models can improve understanding of prediction reliability.
